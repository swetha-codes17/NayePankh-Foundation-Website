
# NayePankh Foundation Website

This project is a responsive and user-friendly website developed for NayePankh Foundation using HTML, CSS, and JavaScript.

The website aims to promote the foundation's mission of empowering students through internships, skill development programs, mentorship, and career opportunities.

Features:
- Responsive Design
- Modern Navigation Bar
- Hero Section
- About Us Section
- Internship Programs
- Why Choose NayePankh Section
- Success Stories
- Contact Form
- Footer
- Hover Effects and Animations
- Dark Mode Support
- Smooth Scrolling Navigation

The project focuses on providing a professional and engaging user experience while showcasing the various opportunities offered by NayePankh Foundation. The website is designed with a clean interface, attractive layout, and interactive elements to improve accessibility and user engagement.

Technologies Used:
- HTML5
- CSS3
- JavaScript

This project was created as part of the NayePankh Foundation Internship Selection Assignment.
